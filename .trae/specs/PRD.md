## 一、产品概述

### 1.1 产品定位

《非遗造物局》是一个面向年轻人的 **AI 非遗文创与手作方案生成平台**。通过将剪纸、皮影、苗绣、扎染、木版年画等非遗元素，与哆啦A梦、熊大熊二等现代流行IP结合，转译为拼豆挂件、手机壳、帆布包、贴纸等现代文创产品方案，并通过 **Three.js 3D视觉展示**呈现效果，让非遗从"被观看的文化"变成"可以亲手制作、日常使用和社交分享的作品"。

### 1.2 产品目标

- **核心目标**：降低非遗文创设计和手作门槛，让年轻人轻松参与非遗共创
- **创新目标**：通过AI生图和3D展示，实现非遗的年轻化表达与传播
- **商业价值**：帮助文创小店、景区店、博物馆商店快速生成非遗主题商品概念
- **社会价值**：让非遗技艺获得更年轻、更轻量的传播方式

### 1.3 产品形态

- **形态**：Web 网站 / Web App
- **核心体验**：选择非遗技艺 → 选择流行IP → 选择文创载体 → AI生图生成 → 3D视觉展示 → 生成拼豆图纸和商品方案

## 二、目标用户

### 2.1 用户画像

| 用户类型 | 描述 | 核心需求 |
| --- | --- | --- |
| **年轻手作爱好者** | 喜欢国潮、手作、拼豆、贴纸，追求个性化表达的年轻人（18-30岁） | 快速获取非遗主题拼豆图纸，制作有文化感的手作小物 |
| **文创小店店主** | 想做非遗主题商品但缺少设计能力的文创小店、景区店、博物馆商店 | 快速生成商品概念和预览图，降低设计试错成本 |
| **文化传播者** | 非遗传承人、文旅机构、校园社团，希望让非遗更年轻化传播 | 将传统纹样转化为适合年轻人传播的视觉方案 |
| **IP爱好者** | 喜欢哆啦A梦、熊大熊二等流行IP的粉丝群体 | 看到喜欢的IP以非遗形式呈现，产生购买和分享欲望 |

### 2.2 使用场景

1. **日常手作**：年轻人想制作一个有文化感的拼豆挂件、手机壳或贴纸
2. **商品开发**：文创小店想快速生成一批非遗主题商品概念用于上新前验证
3. **活动周边**：校园社团或文旅活动需要快速产出非遗主题周边和宣传物料
4. **文化传播**：传承人希望把传统纹样转成更适合年轻人传播的视觉方案
5. **IP创意**：粉丝希望看到喜欢的IP以非遗形式重新演绎，如哆啦A梦剪纸、熊大熊二年画

### 3.2 功能详细描述

#### 3.2.1 非遗技艺选择

- **功能描述**：用户从下拉菜单中选择一种非遗技艺作为创作灵感来源
- **选项列表**：
  1. 剪纸 - 中国传统民间艺术，以剪刀或刻刀在纸上剪刻花纹
  2. 皮影 - 用兽皮或纸板做成的人物剪影以表演故事的民间戏剧
  3. 苗绣 - 苗族民间刺绣，色彩鲜艳，纹样繁复
  4. 扎染 - 中国民间传统而独特的染色工艺
  5. 木版年画 - 用木版印刷的年画，线条粗犷，色彩鲜明

#### 3.2.2 流行IP选择（新增）

- **功能描述**：用户选择一个流行IP，系统将其与非遗技艺融合，生成非遗风格的IP图像
- **选项列表**：
  1. 哆啦A梦 - 经典日本动漫角色
  2. 熊大熊二 - 国产动画《熊出没》角色
  3. 哪吒 - 中国神话人物，国漫热门角色
  4. 孙悟空 - 中国神话经典角色
  5. 皮卡丘 - 精灵宝可梦热门角色
- **融合效果**：
  - 剪纸风格：IP角色呈现剪纸镂空效果，色彩鲜艳
  - 皮影风格：IP角色呈现半透明皮影效果，可投影
  - 苗绣风格：IP角色身上布满苗绣纹样
  - 扎染风格：IP角色呈现扎染纹理效果
  - 木版年画风格：IP角色呈现年画粗犷线条和鲜明色彩

#### 3.2.3 文创载体选择

- **功能描述**：用户选择非遗纹样的呈现载体
- **选项列表**：
  1. 拼豆挂件 - 用拼豆制作的挂饰
  2. 帆布包 - 印有非遗纹样的帆布包
  3. 手机壳 - 印有非遗纹样的手机壳
  4. 贴纸套组 - 非遗主题贴纸套装
  5. 冰箱贴 - 非遗主题冰箱贴

#### 3.2.4 年轻化风格选择

- **功能描述**：用户选择非遗纹样的呈现风格
- **选项列表**：
  1. 国潮明亮 - 高饱和度配色，现代国潮风格
  2. 可爱校园 - 圆润线条，可爱卡通风格
  3. 复古市集 - 怀旧色调，复古文艺风格
  4. 极简日常 - 简约线条，低饱和度配色
  5. 节日礼物 - 喜庆配色，节日氛围

#### 3.2.5 AI图像生成（新增）

- **功能描述**：使用nanobanana API，根据用户选择的非遗技艺、IP和风格，生成非遗风格的IP图像
- **输入参数**：
  - 非遗技艺类型
  - 流行IP名称
  - 风格描述
- **输出内容**：
  - 非遗风格的IP图像（PNG格式）
  - 图像分辨率：1024×1024
- **API配置**：
  - 使用nanobanana API Key进行认证
  - 支持异步生成，显示加载进度

#### 3.2.6 Three.js 3D视觉展示（新增）

- **功能描述**：将AI生成的图像展示在3D文创载体模型上，支持用户交互
- **3D模型列表**：
  1. 拼豆挂件模型 - 悬挂式挂件，可旋转查看
  2. 帆布包模型 - 立体帆布包，可旋转、缩放
  3. 手机壳模型 - 手机壳套在手机上的效果
  4. 贴纸模型 - 贴纸贴在平面上的效果
  5. 冰箱贴模型 - 贴在冰箱上的效果
- **交互功能**：
  - 鼠标拖拽旋转3D模型
  - 滚轮缩放查看细节
  - 点击切换材质/纹理
  - 自动旋转动画
- **视觉效果**：
  - 真实光照渲染
  - 材质纹理映射
  - 阴影效果
  - 环境反射

#### 3.2.7 商品文案生成

- **功能描述**：系统根据用户选择的非遗技艺、IP、载体和风格，自动生成商品名称、卖点文案和非遗故事卡
- **输出内容**：
  - 商品名称（如：哆啦A梦剪纸拼豆挂件）
  - 卖点文案（如：将经典动漫角色哆啦A梦以剪纸艺术重新演绎，镂空工艺展现传统美学）
  - 非遗故事卡（介绍该非遗技艺的历史背景和文化内涵）

#### 3.2.8 拼豆图纸生成

- **功能描述**：将AI生成的非遗IP图像转译为像素化的拼豆图纸
- **输出内容**：
  - 带坐标网格的拼豆图纸（18×12 格子）
  - 每个格子标注色号（A/B/C/D）
  - 成品挂件预览图

#### 3.2.9 颜色统计与材料清单

- **功能描述**：统计拼豆图纸中每种颜色的使用数量
- **输出内容**：
  - 颜色名称（朱砂红、孔雀绿、鎏金黄、靛青蓝）
  - 色号代码（A-12、C-34、S-08、M-22）
  - 每种颜色所需豆子数量

#### 3.2.10 制作信息展示

- **功能描述**：展示制作该拼豆挂件所需的信息
- **输出内容**：
  - 预计豆子数量
  - 使用颜色数量
  - 预计制作时长
  - 制作难度等级（入门/进阶）

### 4.2 核心交互流程

1. **进入页面**：用户打开网站，默认显示苗绣 + 哆啦A梦 + 拼豆挂件 + 国潮明亮的组合方案
2. **选择参数**：用户依次选择非遗技艺、流行IP、文创载体和风格
3. **生成图像**：点击"生成AI图像"按钮，使用nanobanana API生成非遗风格的IP图像
4. **3D展示**：AI图像生成完成后，自动映射到3D文创载体模型上，用户可旋转、缩放查看
5. **生成图纸**：点击"生成拼豆图纸"按钮，生成带坐标的专业拼豆图纸
6. **查看统计**：查看颜色统计、材料清单和制作信息

## 六、非功能需求

### 6.1 性能需求

- 页面加载时间 < 3 秒（含Three.js加载）
- AI图像生成响应时间 < 10 秒
- 3D模型渲染流畅，无卡顿
- 图案生成响应时间 < 100ms

### 6.2 兼容性需求

- 支持主流浏览器（Chrome、Firefox、Safari、Edge）
- 支持移动端和桌面端访问
- 响应式布局，适配不同屏幕尺寸
- 支持WebGL 2.0（Three.js依赖）

### 6.3 可用性需求

- 界面简洁直观，操作流程清晰
- 关键按钮尺寸足够，便于触控操作
- 文案清晰易懂，无专业术语障碍
- AI生图过程显示进度条，减少等待焦虑

### 6.4 安全性需求

- nanobanana API Key存储安全，不暴露在前端代码中
- 图像生成请求使用安全的API调用方式
